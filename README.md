# Entrepreneurship App Project

A Flutter project that displays entrepreneurship course content and provides RAG-based question-answering functionality.

## Description

This project contains course content that will be used in a Flutter app. The app will:
- Display the course content to users
- Implement RAG (Retrieval-Augmented Generation) based Q&A to answer questions from the course content

## Project Structure

```
├── course/              # Course content (15 modules)
│   ├── 01_finding_opportunities/
│   ├── 02_entrepreneurial_reality/
│   ├── 03_founder_mindset/
│   └── ...
├── instructions/        # Content curation instructions (ignored in git)
└── README.md
```

## Course Content

The `course/` directory contains 15 modules covering various entrepreneurship topics:
- Finding Opportunities
- Entrepreneurial Reality
- Founder Mindset
- Product Development
- Execution
- Strategy
- Building Teams
- Growth
- Fundraising
- And more...

All course content is in markdown format and ready to be integrated into the Flutter app.
